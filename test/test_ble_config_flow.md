# BLE 配网手工测试矩阵

| 用例ID | 输入 | 预期 |
|---|---|---|
| TC-01 | 正确 6 位码 | 返回授权成功 |
| TC-02 | 错误 6 位码 | 返回 `1002` |
| TC-03 | 过期 6 位码 | 返回 `1003` |
| TC-04 | 空 SSID | 返回 `2002` |
| TC-05 | NTP 服务器空 | 返回 `2002` |
| TC-06 | 错误 WiFi 密码 | 返回 `3001` |
| TC-07 | 无效 NTP 地址 | 返回 `3002` |
| TC-08 | 长按按键重入 | 重新广播并显示新配网码 |
| TC-09 | 旧 AP 页面 | 不可访问（已移除） |
